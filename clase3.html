¿Qué aprenderemos hoy?

Aprenderemos a consolidar datos

Vamos a contar cuántos clientes tenemos en la tabla clientes 


SELECT * FROM clientes;
Resultado: 122

Usemos una consulta más adecuada


SELECT COUNT(*) from clientes;

Obtener la suma de toda una columna 


SELECT SUM(cantidad) from pagos;

Lo podemos combinar con filtros:


SELECT SUM(cantidad) from pagos WHERE fechaPago = '2003-07-21';

También podemos obtener el pago Minimo y el Máximo 


SELECT MIN(cantidad) from pagos;

Obtener registros únicos 


SELECT DISTINCT(ciudad) from clientes;
